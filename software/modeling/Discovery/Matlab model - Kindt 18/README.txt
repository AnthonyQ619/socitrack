Analytical Model for computing the discovery latency of duty-cycled protocols.
2015-2017 Philipp Kindt <kindt@rcs.ei.tum.de>

This matlab code implements the algorithm presented in the paper

Philipp H. Kindt, Marco Saur, Michael Balszun and Samarjit Chakraborty.
Neighbor Discovery Latency in BLE-Like Protocols.
IEEE Transactions on Mobile Computing (TMC), to appear

*How to use*:
See createFigure.m as an example. This matlab script will create a plot over multipe advertising intervals.

*More Information*
Please visit www.rcs.ei.tum.de/wireless-systems for updates and for a preprint of the paper!
